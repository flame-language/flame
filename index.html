Flame

What is Flame?

Flame is a new statically-typed programming language that borrows concepts from other popular languages, such as:

  • Java
  • VB
  • Python

How do I use Flame?

For now Flame is under development, it's parser has been built through the use of JavaCC meanwhile it's interpreter is still being built. At the moment, you can download Flame, compile it and type non-sensical ramblings into your console.

Can I be involved?

Sure! Flame is an OSS language and is free for anyone to use, modify and contribute to.
